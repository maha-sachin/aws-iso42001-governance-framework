# Visualization Layer

The visualization layer presents the governance framework as an enterprise AI platform experience.

## Module 1 - Dashboard

Displays:

- Compliance Score
- Violations
- Deployment Status
- Risk Level

## Module 2 - Corporate AI Governance

Displays:

- Responsible AI
- Data Privacy
- Transparency
- AI Security

## Module 3 - AI Lifecycle Governance

Displays lifecycle controls across:

- Scoping
- Data Collection
- Training
- Validation
- Deployment
- Monitoring
- Retirement

## Module 4 - Policy Engine

Displays:

- OPA
- Rego
- ISO 42001 Controls

## Module 5 - CI/CD Governance Gate

Displays:

- GitHub Actions
- Policy Evaluation
- Deployment Approval

## Module 6 - Compliance Reports

Displays:

- Violations
- Remediation
- Audit Evidence

## Run the Dashboard

```bash
npm install
npm run dashboard
```

Then open the Vite URL shown in the terminal, usually `http://localhost:5173`.
